Project Overview

This template demonstrates the integration of Rust and WebAssembly for high-performance web applications. It allows you to write computationally intensive code in Rust that runs directly in the browser via WebAssembly.

Features

  • Rust code compiled to WebAssembly for near-native performance
  • Seamless JavaScript integration
  • Modern development workflow with hot reloading
  • Built-in testing infrastructure
  • Optimized for production builds

Example Function

This template includes a simple greeting function:

  • greet(name): Generates a personalized greeting message
